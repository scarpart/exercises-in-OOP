Code for an object-oriented card game, the rules of which are as follows:

1. It's a game for two players.
2. Each of the players has 5 cards.
3. The cards contain attributes and the character that each of them are associated with.
4. The players will put the card on the table on each round. If the sum of the attributes of one player's card beats the other, the former takes both cards into their own hand.
5. If the result is a draw, then each player gets their card back.
6. Loses the one who has no cards left.
